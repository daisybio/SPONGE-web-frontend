Gene Betweenness centrality focuses on bottlenecks. A node with high betweenness centrality within a graph lies on a significant proportion of the shortest paths between two other nodes. {{ node.betweenness }} Eigenvector centrality is a measure of a node's influence in a network. A node with high eigenvector centrality is pointed to by many nodes, which also have high eigenvector centrality. {{ node.eigenvector }} Degree centrality measures the number of edges connected to a node and emphasizes the significance of hub nodes, which are recognized as pivotal elements within biological systems. {{ node.node_degree }}

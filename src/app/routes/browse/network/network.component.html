What you see is a subset of a large ceRNA network obtained by applying SPONGE to data from TCGA. The entire network is too large to visualize here. You can adjust the filter criteria using the controls on the left.

The size of the nodes is proportional to the degree of the node. The thickness of the edges is proportional to the mscor value of the interaction.

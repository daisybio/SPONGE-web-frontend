What you see is a subset of a large ceRNA network obtained by applying SPONGE to data from TCGA. The entire network is too large to visualize here. You can adjust the filter criteria using the controls on the left.

Shapes

Circle Node, where a corresponding entry is present in the opposite network. If you see a circle in the gene network, there is a corresponding node in the transcript network with the same filter criteria. The same applies vice versa.
Square Node, where no corresponding entry is present in the opposite network. If you see a square in the gene network, there is no corresponding node in the transcript network with the same filter criteria. The same applies vice versa.

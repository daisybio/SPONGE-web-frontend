You can enter a gene symbol or ENSG number to search in the SPONGE database for all corresponding interactions. On the next side you will find tables of the found cancer sets. You can search those tables for other genes or create a network from those table entries by clicking the "Show as Network" Button. You can also select some entries and export only those to the network.

Browse Tab

Here you can browse ceRNA interaction networks for different diseases. The nodes correspond to genes or transcripts, whereas the edges correspond to the miRNAs the neighboring nodes compete over. The node size is corresponding to node degree and edge thickness is corresponding to mscor. The node shape in a transcript/gene-level network is a circle or a square depending on whether the corresponding node is found on the corresponding gene/transcript-level network with the same filtering criteria.
